# Design folder — Sky Defender Remaster

Материалы дизайна ремастера Flash-игры **Sky Defender: Joe's Story** (2010–2011).

**Начинать отсюда → [remaster-spec.md](remaster-spec.md)** — спецификация разработки:
что за игра, ядро геймплея, маппинг на движок pix3, план по фазам M0–M5 и указатель
на все остальные документы.

## Что где

| Папка | Содержимое |
|---|---|
| `gdd/` | Оригинальный GDD v1.12 (RU), 30 миссий с диалогами, брифинги, тексты UI/магазина/ачивментов, интро, заметки по анимации Джо и наградам — извлечено из авторских .doc 2010–2011 |
| `original-data/` | Игровые конфиги оригинала: `conf.xml` (баланс оружия/мобов/магазина), `mobs.xml` (волны), `units.txt` (реестр 48 юнитов), `presets.txt` (паттерны построений), `support1.xml`, `sky.xml` (параллакс облаков) |
| `references/` | `screens/` — скриншоты оригинала (магазин, статистика); `classification/` — авторские схемы классов юнитов и боссов; `schemes/` — схемы вооружения врагов |

## Происхождение

Исходные .fla Flash-версии не сохранились. Механика восстановлена из GDD, внешних
XML-конфигов и декомпиляции релизной сборки `cannon_game_v10.18.swf` (JPEXS FFDec) —
семантика формата волн и баллистика подтверждены по коду. Игровые ассеты — финальная
итерация `resources24` + релизный набор звуков `soundres` из архива проекта.
