# Design folder

Put all your game design materials here — both the document and the pictures,
in one place:

- **Game design document** — `gdd.md` / `gdd.pdf` / any format. Describe the core
  loop, controls, win/lose conditions, screens, and progression.
- **Reference images** — screenshots of games you like, mood boards, sketches,
  palettes, UI mockups. Drop them straight into this folder.

AI agents working on this project read this folder first and build the game from
what they find here. The more concrete the GDD (numbers, screen lists, examples),
the closer the first playable version will be to what you imagined.
