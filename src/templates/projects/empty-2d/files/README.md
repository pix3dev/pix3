# {{PROJECT_NAME}}

A Pix3 project created from the **Empty 2D** template.

## Project structure

- `design/` — game design document and reference images
- `scenes/` — scene files (`.pix3scene`, YAML)
- `sprites/` — images and textures
- `scripts/` — game scripts (`export class X extends Script`, used as `user:X`)
- `audio/` — music and sound effects
- `AGENTS.md` — rules and pointers for AI agents working on this project

New asset-type folders (models, fonts, …) are created as you add those assets.

## Getting started

1. Open the project folder in the Pix3 editor.
2. `scenes/main.pix3scene` is the startup scene — press Play to run it.
3. Drop your GDD and reference art into `design/`, then ask an AI agent to build the game.
